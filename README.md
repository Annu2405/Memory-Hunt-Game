# 🎮 Memory Hunt Game

This is a simple 2-player memory matching game made using Python.
The main idea of the game is to match pairs of images and score points.

---

## About the Project

This project is built using Python with a graphical interface.
It was created as a mini project to understand GUI programming and basic game logic.

Players take turns selecting tiles. If the tiles match, the player gets a point. Otherwise, the turn switches to the other player.

At the end, the player with the highest score wins.

---

## Features

* 2-player game
* Simple GUI using Tkinter
* Randomly shuffled tiles
* Score tracking
* Move counter
* Result screen with winner
* Bar graph showing scores
* Start and end sound effects

---

## Technologies Used

* Python
* Tkinter
* PIL (for images)
* Matplotlib
* Pygame (for sound)

## How to Play

1. Enter player names
2. Click on tiles to reveal images
3. Match pairs to score points
4. If tiles don’t match, turn changes
5. Game ends when all pairs are matched
